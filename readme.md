
### Installation
```bash
composer require sustartx/laravel-sms-providers
```
`.env` içinde aşağıdaki değişiklikleri yapın.
```dotenv
SMS_USERNAME=
SMS_PASSWORD=
SMS_URL=
SMS_TOKEN=
SMS_COUNTRY_PREFIX=
SMS_FROM=
```
